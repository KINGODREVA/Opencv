## 파이썬 가상환경설정

```
파이썬 버전확인
- $ pyton3 -V

pip 버전확인
- $ pip3 -V

가상 환경 설치 ( 파이썬버전에 맞춰서 ex 3.8)
- $ sudo apt install python3.8-venv

가상환경 생성 python3 -m venv [가상환경이름]
- $ python3 -m venv venv

가상환경 활성
- linux or mac
- $ source ./venv/bin/activate

가상환경 확인
- $ (venv)$ python -V

가상환경 비활성
- $ (venv)$ deactivate
```



## 가상환경 내에 opencv 설치

```
- $ source ./venv/bin/activate
- (venv) $ pip install opencv-contrib-python

pip 설치 내용 확인(둘 중 하나)
- (venv) $ pip list
- (venv) $ pip freeze

pip 의존성패키지를 내보내기
- (venv) $ pip freeze > requirements.txt

pip 의존성패키지 복원(폴더 명시)
- (venv) $ pip install -r requirements.txt

특정 버전으로 설치할때
- pip install opencv-contrib-python== __Version__(ex: 4.7.0.72)

```

## 예습

```python

import cv2

img_file = "yeosu.jpg"
img = cv2.imread(img_file)

# is는 타입비교
if img is not None:
    cv2.imshow("IMG", img)
    
    # 아무런 값을 남기지 않으면 무한대기
    cv2.waitKey()
    
    # 모든 윈도우 제거
    cv2.destroyAllWindows()
    pass
else:
    print("no image file.")

```
